Lección 1

¿Qué es Python?

Python es un lenguaje de programación creado para que escribir código sea algo claro, ordenado y fácil de leer, casi como escribir en inglés sencillo. Por eso es el lenguaje favorito para empezar a programar.

Un poco de historia

Python fue creado por Guido van Rossum, un programador neerlandés, y se publicó por primera vez en el año 1991. Guido quería un lenguaje que fuera divertido de usar (de hecho, el nombre "Python" viene del programa de comedia británico "Monty Python's Flying Circus", no de la serpiente).

Desde entonces, Python ha crecido muchísimo y hoy es uno de los lenguajes de programación más usados en el mundo, tanto por principiantes como por grandes empresas como Google, Netflix, Spotify e Instagram.

¿Para qué sirve Python?

Python sirve para crear casi cualquier tipo de programa: páginas web, aplicaciones, juegos, herramientas que automatizan tareas repetitivas, y programas que analizan grandes cantidades de información. Su sintaxis simple permite que una persona sin experiencia pueda escribir su primer programa en minutos.

¿Dónde se utiliza Python?

Estas son algunas de las áreas donde más se usa Python hoy en día:

🌐

Desarrollo web

Con herramientas como Django o Flask, Python permite crear sitios y aplicaciones web completas, desde el lado del servidor.

🤖

Inteligencia artificial

Python es el lenguaje preferido para crear modelos de inteligencia artificial y aprendizaje automático, gracias a librerías como TensorFlow y PyTorch.

📊

Ciencia de datos

Permite analizar, ordenar y visualizar grandes cantidades de datos para encontrar patrones y tomar mejores decisiones.

⚙️

Automatización

Ideal para crear scripts que hacen tareas repetitivas de forma automática, como organizar archivos o enviar correos.

🎮

Videojuegos

Con librerías como Pygame se pueden crear videojuegos sencillos en 2D, ideales para aprender lógica de programación.

Lección 2

Variables

Una variable es como una caja con una etiqueta donde guardamos información para usarla más adelante. Le damos un nombre a esa caja y dentro guardamos un valor: un número, un texto, etc.

¿Cómo se declara una variable?

En Python, declarar una variable es muy sencillo: solo escribes el nombre que quieras darle, el signo = (que significa "asignar", no "es igual a") y el valor que quieres guardar.

No necesitas indicar de qué tipo es la variable. Python lo detecta automáticamente según el valor que le das.

Tipos de datos básicos

  • int Números enteros, sin decimales. Ejemplo: 20
  • float Números con decimales. Ejemplo: 1.75
  • string Texto, siempre entre comillas. Ejemplo: "Juan"
  • bool Valores de verdadero o falso. Ejemplo: True
Ejemplo: Variables en Python 
nombre = "Juan"
edad = 20
altura = 1.75
estudiante = True

print(nombre)
print(edad)
print(altura)
print(estudiante)

💡 Explicación línea por línea:

Las primeras cuatro líneas crean variables: nombre guarda el texto "Juan", edad guarda el número 20, altura guarda el número decimal 1.75, y estudiante guarda el valor booleano True (verdadero). Las últimas cuatro líneas usan print() para mostrar en pantalla el contenido de cada variable.

Lección 4

Condicionales

Los condicionales permiten que un programa tome decisiones. Con if, elif y else, le decimos a Python: "si se cumple esto, haz esto; si no, haz aquello".

¿Cómo funciona?

  • if — se ejecuta si la condición es verdadera.
  • elif — significa "si no, y si esto otro es verdadero..." (puede haber varios).
  • else — se ejecuta si ninguna condición anterior fue verdadera.

Algo muy importante en Python: el bloque de código dentro de un if debe estar indentado (con espacios o tabulación). Esa indentación es lo que le indica a Python qué líneas pertenecen al bloque.

Inicio
↓
¿edad >= 18?
Sí
↓
Imprimir "Eres mayor de edad"
No
↓
Imprimir "Eres menor de edad"
↓
Fin
Ejemplo: Condicional if / else 
edad = 18

if edad >= 18:
    print("Eres mayor de edad")
else:
    print("Eres menor de edad")
El resultado aparecerá aquí...
💡 Explicación línea por línea:

La línea 1 crea la variable edad con el valor 18. La línea 3 pregunta: "¿edad es mayor o igual a 18?". Como 18 es igual a 18, la condición es True, así que se ejecuta la línea 4 y se imprime "Eres mayor de edad". Si la edad hubiera sido menor a 18, se habría ejecutado el bloque else en su lugar.

Lección 5

Ciclos (bucles)

Los ciclos permiten repetir una acción varias veces sin tener que escribir el mismo código una y otra vez. Python tiene dos tipos principales: for y while.

Ciclo for

El ciclo for se usa cuando sabemos exactamente cuántas veces queremos repetir algo, o cuando queremos recorrer los elementos de una lista, texto, etc.

Ejemplo: Ciclo for 
for i in range(5):
    print(i)
El resultado aparecerá aquí...
💡 Explicación línea por línea:

range(5) genera una secuencia de números del 0 al 4 (5 números en total, pero empezando desde 0). La variable i toma, uno por uno, cada valor de esa secuencia: primero 0, luego 1, luego 2, y así hasta 4. En cada repetición, print(i) muestra el valor actual de i.

Ciclo while

El ciclo while repite un bloque de código mientras una condición sea verdadera. Es útil cuando no sabemos de antemano cuántas veces se va a repetir.

Ejemplo: Ciclo while 
contador = 0

while contador < 5:
    print(contador)
    contador += 1
El resultado aparecerá aquí...

🔄 Así cambia la variable "contador" en cada repetición:

contador = 0 → se imprime 0 → contador += 1
contador = 1 → se imprime 1 → contador += 1
contador = 2 → se imprime 2 → contador += 1
contador = 3 → se imprime 3 → contador += 1
contador = 4 → se imprime 4 → contador += 1
contador = 5 → la condición (5 < 5) es False → el ciclo termina
💡 Explicación paso a paso:

contador = 0 crea la variable con valor inicial 0. La condición contador < 5 se revisa antes de cada repetición. Mientras sea verdadera, se ejecuta el bloque: se imprime el valor actual y luego contador += 1 le suma 1. Cuando contador llega a 5, la condición se vuelve falsa y el ciclo se detiene.

Lección 6

Funciones

Una función es un bloque de código reutilizable que realiza una tarea específica. En lugar de repetir el mismo código varias veces, lo escribimos una vez dentro de una función y la "llamamos" cuando la necesitamos.

Partes de una función

  • def — palabra que indica que vamos a crear (definir) una función.
  • Nombre — el nombre que le damos a la función, como suma.
  • Parámetros — valores que la función recibe para trabajar, escritos entre paréntesis.
  • return — palabra que indica el valor que la función devuelve como resultado.
Ejemplo: Función suma 
def suma(a,b):
    return a+b
resultado = suma(5,3)
print(resultado)

Lección 7

Listas

Una lista es una colección ordenada de elementos guardados en una sola variable. Se escriben entre corchetes [ ] y los elementos se separan con comas.

Ejemplo: Crear una lista 
frutas = ["manzana","pera","uva"]
print(frutas)

Funciones más usadas con listas

➕

append()

Agrega un elemento al final de la lista.

📍

insert()

Agrega un elemento en una posición específica.

🗑️

remove()

Elimina la primera coincidencia de un valor.

📤

pop()

Elimina y devuelve el último elemento (o uno por posición).

📏

len()

Devuelve cuántos elementos tiene la lista.

Lección 8

Diccionarios

Un diccionario guarda información en pares de clave : valor, como una ficha de datos. En lugar de acceder a los elementos por su posición (como en una lista), accedemos a ellos por su clave.

Ejemplo: Crear un diccionario 
persona = {
    "nombre": "Carlos",
    "edad": 20,
    "ciudad": "México"
}

Claves y valores

En el ejemplo anterior, "nombre", "edad" y "ciudad" son las claves. "Carlos", 20 y "México" son sus valores correspondientes.

Operaciones comunes

  • Acceder: persona["nombre"] devuelve "Carlos".
  • Agregar: persona["pais"] = "México" añade una nueva clave.
  • Eliminar: del persona["edad"] elimina esa clave y su valor.
  • Recorrer: con un ciclo for clave in persona: podemos pasar por todas las claves.
